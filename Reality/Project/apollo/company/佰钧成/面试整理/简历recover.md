## Git 

1. 什么是git flow

   git flow是一种git的使用规范，具体指

   1. master分支永远处于最稳定的状态，保证可运行
   2. dev分支用于日常开发，稳定后merge到release分支
   3. release分支基于dev分支，用于预发布
   4. feature分支用于特别功能的开发

   master相当于游戏的稳定版，release相当于内测，dev相当于公测
